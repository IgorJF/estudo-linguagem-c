# 📚 Exercícios de Programação 1 — Linguagem C

Repositório com exercícios resolvidos da disciplina de **Programação 1**, desenvolvidos em linguagem **C**.

Os exercícios estão organizados em 7 listas (Lista I a Lista VII), cobrindo os tópicos introdutórios da programação estruturada.

---

```
exerciciosprogr1/
├── EXERCICIOS LISTA I/
├── EXERCICIOS LISTA II/
├── EXERCICIOS LISTA III/
├── EXERCICIOS LISTA IV/
├── EXERCICIOS LISTA V/
├── EXERCÍCIOS LISTA VI/
└── EXERCÍCIOS LISTA VII/
```

---

## 🛠️ Como compilar

```bash
gcc nome_do_arquivo.c -o programa
./programa
```

> Requer GCC instalado. No Windows, você pode usar [MinGW](https://www.mingw-w64.org/) ou [WSL](https://learn.microsoft.com/pt-br/windows/wsl/).
> Ao utilizar WSL, gcc já instalado.
